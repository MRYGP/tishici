## 📊 学习方法

### 核心原则
1. **案例驱动**：从具体案例出发，不空谈理论
2. **模型提炼**：提炼可复用的分析框架和决策模型
3. **洞察跃迁**：从具体到抽象，从案例到规律
4. **实战应用**：所有学习成果都要应用到实际决策

### 学习流程

```
案例拆解（Why/Money/What/How/Growth）
    ↓
模型提炼（变量体系、因果链条、适用边界）
    ↓
洞察归纳（跨案例规律、行业趋势、决策原则）
    ↓
实战应用（业务决策、战略规划、风险判断）
```
