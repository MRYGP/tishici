# 按企业生命周期分类

本目录按照企业所处的生命周期阶段对案例进行分类存储。

---

## 📁 子目录

### 创业期/
0-1阶段，寻找产品市场匹配（PMF）的案例

### 成长期/
1-10阶段，快速扩张和规模化的案例

### 成熟期/
维持市场地位、优化运营效率的案例

### 转型期/
面临行业变革、主动或被动转型的案例

---

## 📝 分类说明

**创业期特征**：
- 验证商业模式
- 寻找早期客户
- 快速迭代产品

**成长期特征**：
- 快速扩张市场
- 建立竞争壁垒
- 完善组织架构

**成熟期特征**：
- 市场地位稳固
- 追求运营效率
- 探索第二增长曲线

**转型期特征**：
- 面临行业颠覆
- 调整战略方向
- 重构商业模式
