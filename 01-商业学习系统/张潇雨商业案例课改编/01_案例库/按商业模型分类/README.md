# 按商业模型分类

本目录按照商业模型对案例进行分类存储。

---

## 📁 子目录

### 平台模式/
双边市场、多边平台等平台型商业模式案例

### 订阅模式/
会员制、订阅制、SaaS等订阅型商业模式案例

### 飞轮效应/
通过正反馈循环实现增长的商业模式案例

### 网络效应/
用户越多价值越大的网络效应型商业模式案例

---

## 📝 使用说明

同一个案例可能体现多种商业模型，建议：
1. 在最核心的模型目录下存放原文件
2. 在其他相关模型目录下创建软链接或引用

示例：
- Amazon的飞轮效应案例存放在 `飞轮效应/` 目录
- 同时在 `平台模式/` 目录下添加引用
