# 按行业分类

本目录按照行业对案例进行分类存储。

---

## 📁 子目录

### 科技互联网/
互联网平台、软件服务、硬件制造等科技类企业案例

### 消费零售/
零售、电商、消费品、餐饮等消费类企业案例

### 金融/
银行、保险、投资、支付等金融类企业案例

### 传统行业转型/
制造业、能源、交通等传统行业的数字化转型案例

---

## 📝 命名规范

文件命名格式：`[企业名]-[案例关键词].md`

示例：
- `Netflix-从DVD到流媒体.md`
- `Amazon-飞轮效应的实践.md`
- `特斯拉-垂直整合战略.md`
