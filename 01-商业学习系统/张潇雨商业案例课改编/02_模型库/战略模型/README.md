# 战略模型

本目录存放从案例中提炼的战略层面商业模型。

---

## 📝 什么是战略模型？

战略模型关注企业的长期竞争优势和战略定位，包括：
- 如何选择市场和赛道？
- 如何建立竞争壁垒？
- 如何应对行业变革？

---

## 🏗️ 常见战略模型

### 经典模型
- **波特五力模型**：行业竞争分析
- **蓝海战略**：开创新市场空间
- **破坏性创新**：用低端市场颠覆高端
- **护城河理论**：建立持久竞争优势

## 📋 模型文档格式

每个模型文档应包含：
1. **模型名称**
2. **核心逻辑**：一句话说清楚
3. **变量体系**：关键变量和因果关系
4. **适用条件**：什么情况下有效
5. **支撑案例**：哪些案例验证了这个模型
6. **决策应用**：如何用于实际决策
