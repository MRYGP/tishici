# 财务模型

本目录存放从案例中提炼的财务层面商业模型。

---

## 📝 什么是财务模型？

财务模型关注企业的盈利机制和财务健康度，包括：
- 如何赚钱？（收入模式）
- 赚多少？（利润率）
- 可持续吗？（现金流）

---

## 🏗️ 常见财务模型

### 经典模型
- **单位经济模型**：单个客户的盈利能力
- **LTV/CAC模型**：客户终身价值 vs 获客成本
- **三级火箭模型**：免费引流→中端盈利→高端变现
- **剃刀-刀片模型**：低价硬件+高价耗材

## 📋 模型文档格式

每个模型文档应包含：
1. **模型名称**
2. **核心逻辑**：一句话说清楚
3. **变量体系**：关键变量和因果关系
4. **适用条件**：什么情况下有效
5. **支撑案例**：哪些案例验证了这个模型
6. **决策应用**：如何用于实际决策
