# 运营模型

本目录存放从案例中提炼的运营层面商业模型。

---

## 📝 什么是运营模型？

运营模型关注企业的日常运营机制和增长引擎，包括：
- 如何获取和留存客户？
- 如何提高运营效率？
- 如何实现规模化增长？

---

## 🏗️ 常见运营模型

### 经典模型
- **飞轮效应**：正反馈循环驱动增长
- **网络效应**：用户越多价值越大
- **病毒式增长**：用户自传播
- **规模经济**：规模越大成本越低

## 📋 模型文档格式

每个模型文档应包含：
1. **模型名称**
2. **核心逻辑**：一句话说清楚
3. **变量体系**：关键变量和因果关系
4. **适用条件**：什么情况下有效
5. **支撑案例**：哪些案例验证了这个模型
6. **决策应用**：如何用于实际决策
