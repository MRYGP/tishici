# 行业趋势

本目录存放从案例中提炼的行业演进趋势和未来预测。

---

## 📝 什么是行业趋势洞察？

行业趋势洞察是通过分析同一行业的多个案例，发现的：
- **演进规律**：行业如何发展变化
- **关键转折点**：什么触发了行业变革
- **未来方向**：可能的发展路径

---

## 🏗️ 趋势类型

### 技术驱动类
新技术如何改变行业格局

示例：
- "云计算如何重塑软件行业"
- "AI对金融行业的三个冲击"

### 商业模式类
商业模式的演进趋势

示例：
- "零售业的全渠道融合趋势"
- "从产品思维到服务思维的转变"

### 用户需求类
用户需求变化如何推动行业变革

示例：
- "Z世代消费习惯对零售的影响"
- "远程办公对企业服务的影响"

---

## 📋 趋势文档格式

文件命名：`趋势[编号]-[行业]-[标题].md`

每个趋势文档应包含：
1. **趋势标题**：一句话说清楚
2. **支撑案例**：至少5个相关案例
3. **演进路径**：从过去到现在到未来
4. **关键驱动力**：什么在推动这个趋势
5. **预测与验证**：未来可能如何发展
