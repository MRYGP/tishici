## 🚀 快速开始（3步上手）

### Step 1：了解系统
阅读 `00_系统指令/README.md`，理解Claude-Cursor协作模式

### Step 2：开始第一个案例
1. 向Claude提供案例信息（名称、来源、关键信息）
2. Claude自动执行七步拆解法
3. Cursor根据Claude的指令生成标准化文档
4. 保存到GitHub对应目录

### Step 3：持续积累
- 每周拆解2-3个案例
- 每月提炼1-2个商业模型
- 每季度归纳2-3个跨案例洞察

## 🧠 核心方法论

### 七步拆解法（案例分析）
1. **时空还原**：还原案例发生的背景和环境
2. **问题诊断**：识别核心问题和深层原因
3. **决策拆解**：拆解关键决策的逻辑和权衡
4. **执行拆解**：分析资源调动和执行策略
5. **模型提炼**：提炼核心商业模型和变量
6. **成败分析**：量化结果和关键成功要素
7. **迁移应用**：提炼可迁移的决策框架

### 模型提炼法（模型构建）
1. **模式识别**：识别案例背后的商业逻辑
2. **变量体系**：构建输入-处理-输出-反馈变量
3. **因果链条**：梳理完整的因果关系
4. **边界界定**：明确适用条件和局限性
5. **决策转化**：转化为可执行的决策框架

### 洞察归纳法（规律发现）
1. **主题聚类**：将相关案例分组
2. **对比分析**：识别成功与失败的差异
3. **规律提炼**：提炼跨案例的深层规律
4. **反例验证**：用反例检验洞察的边界
5. **预测验证**：用新案例验证洞察
6. **应用框架**：转化为决策工具
