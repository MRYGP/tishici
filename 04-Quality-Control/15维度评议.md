## 🎯 接力式评议流程设计

### 核心理念

- **单窗口单任务**: 每个对话窗口只执行一个专业维度的评议
- **进度传承**: 每次评议完成后更新此任务书，传递给下一窗口
- **累积完善**: 15个维度评议完成后，论文达到发表就绪状态

### 操作流程

```
用户 → 发送任务书给窗口1 → 执行评议A → 更新任务书 → 
用户 → 发送更新后任务书给窗口2 → 执行评议B → 更新任务书 → 
... → 完成全部15个维度评议
```

1. **跨尺度分析方法**: Einstein-Kodak认知固化模式对比

3. **统一解释理论**: 为现有认知理论提供WHY层面的统一解释
4. **可证伪预测体系**: 明确的实验验证路径设计
