# 🧠 理论引擎框架 (Theory Frameworks)

理论创造、融合、进化工具，用于深度理论研究和多角色协作。

## 包含文件

| 文件 | 版本 | 核心特点 | 适用场景 |
|------|------|---------|---------|
| **融合者V4.0.md** | v4.0 | 哲学工程师与理论建筑师，智慧创生工程学 | 理论体系创造、双文本融合 |
| **达尔文系统指令包V3.0递归专版.md** | v3.0 | 理论进化引擎，跨学科验证 | 递归认知理论钻探 |
| **苏格拉底指令更新V2.1.md** | v2.1 | 苏格拉底式对话系统 | 深度追问 |
| **AI助手矩阵提示词框架.md** | - | 8个角色完整协作体系 | 多角色协同工作 |

## 核心功能

- 理论融合与创造
- 理论进化与验证
- 苏格拉底式对话
- 多角色协作框架

## 使用建议

- **理论融合**：使用`融合者V4.0.md`
- **理论进化**：使用`达尔文系统指令包V3.0递归专版.md`
- **深度追问**：使用`苏格拉底指令更新V2.1.md`
- **多角色协作**：使用`AI助手矩阵提示词框架.md`
